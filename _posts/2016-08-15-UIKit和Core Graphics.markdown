##  绘图

iOS的绘图框架有

1. UIKit

- UIKit 是最高级的界面,他控制视图的布局、组成、绘图、图片、字体、动画等，所有 UI 组件都是 UIKit 的元素

2. Core Graphics（Quartz)

- UIKit 主要的绘图系统，用来绘制自定义视图

3. Core Animation

- Core Animation 提供了强大的动画效果

4. Core Image

- 图片处理

5. OpenGL ES

- OpenGL 的子集  兼容多个平台
